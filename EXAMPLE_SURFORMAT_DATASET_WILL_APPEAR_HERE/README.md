# SurLab dataset output

The converter writes SurLab session folders and dataset-level CSVs here.

Each session folder uses `{animal_ID}_{session_ID}/` with quartet files per stream (`*_data.npz`, `*_timestamps.npz`, `*_schema.json`, `*_metadata.csv`).

See `example_past_converter/EXAMPLE_SURFORMAT_DATASET_WILL_APPEAR_HERE/` for small example schema/metadata files.
